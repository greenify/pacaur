Pacaur

A fast workflow AUR helper using cower as backend

Download this project as a .zip file Download this project as a tar.gz file

Philosophy

pacaur is an Arch User Repository (AUR) helper aiming at speed and simplicity, designed to minimize user prompt interaction and to use an uncluttered interface.

Dependencies

  • cower, a minimal AUR helper
  • expac, a pacman database extraction utility
  • sudo, for privileges management
